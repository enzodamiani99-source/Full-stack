Introdução

C++ (em português: lê-se "cê mais mais", em inglês lê-se "see plus plus") é uma linguagem de programação compilada multi-paradigma com suporte genérico, orientada a objetos e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante usada também na academia por seu grande desempenho e base de utilizadores.

Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems. Em 2008 a Java foi adquirida pela Oracle.

Java foi desenvolvida por um grupo de pesquisadores da SUN Microsystems por volta de 1990. Essa linguagem possui estrutura muito semelhante à linguagem C, da qual descende diretamente.
